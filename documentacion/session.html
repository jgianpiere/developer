Login Redirect usr_prf_lgrd login_redirect

Usaremos la variable de session usr_prf_lgrd para guardar la uri de acceso antes de ser redirigido al formulario de session en caso no este logeado. utilice: $this->session->set_userdata('usr_prf_lgrd',[valor]); . un ejemplo seria el siguiente: url - http://localhost/CI/SOFCA/Compras redirect($this->session->set_userdata('usr_prf_lgrd'),'refresh'); redireccionara al home en caso de no tener url distinta y a dicha url en caso de existir

User Name usr_prf_name user_name

Usaremos la variable de session usr_prf_name para guardar el nombre del usuario. utilice: $this->session->set_userdata('usr_prf_name',[valor]); .
